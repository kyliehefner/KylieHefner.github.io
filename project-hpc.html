Adaptive Mesh Refinement Solver

High Performance Computing Final Project | Chapman University


Overview

This project involved building a C++ solver for the 2D advection-diffusion equation, using a quadtree-based Adaptive Mesh Refinement (AMR) framework. The solver dynamically refines the grid in regions with large error, improving accuracy and efficiency compared to uniform grids.

Objectives

  • Implement AMR from scratch using a quadtree data structure
  • Solve the advection-diffusion equation with finite difference methods
  • Parallelize the computation using OpenMP to reduce runtime
  • Visualize solution accuracy and refinement levels

Technical Details

The solver supports refinement and coarsening based on a threshold applied to local error estimates. Cells are stored in a quadtree, and the PDE is solved iteratively using a finite difference stencil across non-uniform grids. Boundary conditions and flux continuity were carefully handled across levels. Parallel loops were applied using OpenMP for inner PDE solves and error computations.

Results

  • Significant speedup on large grids using OpenMP
  • Improved accuracy compared to fixed-grid solutions for the same number of total points
  • Successfully visualized dynamic refinement levels and evolving solution profiles

Tools & Technologies

  • C++, OpenMP
  • Matplotlib, NumPy
