Hyperbolic Restricted Boltzmann Machine

Master’s Capstone Project – Chapman University


Overview

This project involved developing a Restricted Boltzmann Machine (RBM) that operates in hyperbolic space to better model hierarchical data structures like those found in WordNet.

Objectives

  • Model hierarchical relationships using hyperbolic geometry
  • Compare performance with Euclidean RBMs
  • Explore sampling and training in curved space

Technical Details

The model redefines the RBM’s energy function using Riemannian geometry, and employs custom sampling and optimization routines adapted to hyperbolic space. WordNet noun hierarchies were embedded and reconstructed to evaluate performance.

Results

  • Achieved higher reconstruction accuracy on hierarchical datasets
  • Showed meaningful clustering in hyperbolic latent space

Tools & Technologies

  • Python, NumPy, PyTorch
  • Riemannian optimization, WordNet, Matplotlib
